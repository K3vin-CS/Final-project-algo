Python Space Invaders
A 2D arcade game made with Python and Pygame, using Object-Oriented Programming (OOP).

## Features
- Player movement & shooting
- Alien formations & mystery ship
- Destructible obstacles
- Scoring & high-score persistence
- Sound effects & music

## Requirements
- Python 3.9+
- pygame

## How to Run
1. Install dependencies:
   pip install pygame
2. Run:
   python main.py

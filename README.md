Python Space Invaders
A classic 2D arcade game inspired by Space Invader made with Python and pygame, also with Object-Oriented Programming(OOP). 

## Features
- Player movement & shooting
- Alien formations & mystery ship
- Destructible obstacles
- Scoring & high-score persistence
- Sound effects & music

## Requirements to be able to play
- Python 3.9
- pygame

## Dependencies
1. Install pygame:
   pip install pygame
2. Run:
   python main.py
